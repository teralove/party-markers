# party-markers
Automatically place markers on friendly and enemy classes.

## Commands
### `partymarkers`
- Toggle on/off
- Default is on

## Notes
- Tanks in your party = Yellow
- Healers in your party = Blue
- Healers not in your party = Red
- Client-side, only you can see the markers.
